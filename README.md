## 📖 Description

https://www.tiktok.com/@Stiff_Clips_USA

**AI Shorts Creator** is an automated web application that uses artificial intelligence to analyze long videos (streams, podcasts, webinars, etc.) and automatically extract the most viral moments, converting them into shorts optimized for TikTok, Instagram Reels, and YouTube Shorts.

### What makes this project different?

- ✅ **Intelligent content analysis**: Uses OpenAI's Whisper for transcription and GPT/Claude to identify viral moments
- ✅ **Automatic viral title generation**: Each clip gets an optimized title based on the actual segment content
- ✅ **Perfect 9:16 vertical format**: Videos optimized for social media with stylized subtitles
- ✅ **Split Screen mode for streamers**: Divides the screen to show webcam and content separately
- ✅ **No size limit**: Process videos of any duration
- ✅ **Multi-language**: Support for English, Spanish, and automatic detection

## 🚀 Installation

### Prerequisites

- **Python 3.9+**
- **FFmpeg** (for video processing)
- **OpenAI API Key** (for Whisper + GPT)
- **Anthropic API Key** (optional, for Claude)

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/ai-shorts-creator.git
cd ai-shorts-creator
```

### Step 2: Create Virtual Environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Install FFmpeg

#### Windows:
```bash
# Download from: https://ffmpeg.org/download.html
# Or use Chocolatey:
choco install ffmpeg
```

#### Linux (Ubuntu/Debian):
```bash
sudo apt update
sudo apt install ffmpeg
```

#### macOS:
```bash
brew install ffmpeg
```

### Step 5: Configure Environment Variables

Create a `.env` file in the project root:

```env
# OpenAI API Key (required)
OPENAI_API_KEY=your_api_key_here

# Anthropic API Key (optional, for Claude)
ANTHROPIC_API_KEY=your_api_key_here

# Default AI provider (openai or claude)
AI_PROVIDER=openai

# TikTok Credentials (optional, for auto-publishing)
TIKTOK_USERNAME=your_tiktok_username
TIKTOK_PASSWORD=your_tiktok_password

# Flask
FLASK_ENV=development
MAX_UPLOAD_SIZE=2147483648
```

### Step 6: Run the Application

```bash
python app.py
```

The application will be available at: **http://localhost:3000**

## 📋 Usage

### 1. Upload Video

- Drag and drop a video or click to select
- Supported formats: MP4, AVI, MOV, MKV, WebM
- No size limit

### 2. Configure Options

| Option | Values | Description |
|--------|--------|-------------|
| **AI Provider** | OpenAI / Claude | Model for content analysis |
| **Shorts Duration** | Short / Long | 35s-1min or 1:10-1:30min |
| **Screen Mode** | Normal / Split Screen | Full screen or webcam+content split |
| **Viral Text Language** | Auto / Spanish / English | Language for generated titles |
| **TikTok Publishing** | Yes / No | Auto-publish to TikTok |

### 3. Process Video

- Click on **"🚀 Analyze and Generate Shorts"**
- AI will transcribe the audio
- Analyze the content
- Automatically generate all viral shorts found

### 4. Download or Publish

- Download each short individually
- Copy the optimized text for Instagram/TikTok
- Or automatically publish to TikTok
